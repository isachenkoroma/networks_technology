Листовая структура Android приложения

Android приложение ZIB позволяет реализовать листовую структуру входных данных. В качестве входных данных приложение запрашивает от пользователя ссылку на файл формата json. Для демонстрации примера работы приложения реализована функция использования данных по умолчанию. Данные по умолчанию представляют собой информацию о странах мира, флагах и их столицах. Входной файл имеет вид "ключ-значение". В качестве ключа выступает название объекта (название страны). Значением является подробная информация об объекте (флаг, столица).
